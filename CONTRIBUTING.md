# Contribuindo para o Home Lab 
 
Obrigado por se interessar pela arquitetura deste laboratório! Este é um projeto pessoal focado em estudo prático de infraestrutura, Disaster Recovery e observabilidade. 
 
## Como contribuir 
Se você tem ideias para melhorar os templates do Zabbix, otimizar o dashboard do Grafana ou adicionar novos scripts de automação via PowerShell: 
 
1. Faça um **Fork** deste repositório. 
2. Crie uma branch com a sua feature (`git checkout -b feature/MinhaMelhoria`). 
3. Faça o commit das suas alterações detalhando o que foi feito (`git commit -m 'Add: Novo script de validação de backup'`). 
4. Faça o push para a branch (`git push origin feature/MinhaMelhoria`). 
5. Abra um **Pull Request**. 
 
Toda sugestão de melhoria de arquitetura, correções de documentação ou otimização de scripts será muito bem-vinda e devidamente creditada!
